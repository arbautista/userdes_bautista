First Web Application using Github and Heroku

Reflection Paper No. 3

On the 3rd week of User Design class, we were given a task to create a personal website using HTML and CSS. I have a little background about this because I have created an ecommerce website before when I was in Senior High School. That’s why I haven’t had a hard time working on this task. Github is one of the most popular git version control repository servers. I’ve also been using Github on our Mobile Programming subject and so far, as a student it is a perfect app to host a code. The codes are easy to track because of the commit changes. I’ve used 000webhost before, creating repositories limits only for 2, unlike Github. At first in Github, when I was new beginner at this, the function and purpose of the commit changes are a little bit of confusing but with the help of my classmates, I’ve been able to use it properly. Also, it makes a little harder to use because the there are no search button on repositories. I uploaded my website files and I want to search the index.html, but I needed to overlook it hand by hand. The Heroku app is fast are reliable, all I need to do is to connect my Github account and my chosen repositories. It is easy and reliable to deploy my website and clone my entire website. Overall, I love working with two apps because deploying my website makes me easier and faster.

How would you design an interface for a 1000 floor elevator?

Activity No. 3

Designing a thousand floor for a certain building is pretty hard to design. Afterall, people’s safety is on my hand the moment they stepped in on the elevator or in the building. So, I would make sure the foundation of the building is strong enough to handle a thousand floor building. Next is I would separate the elevator into half, that makes 500-500. I would separate certain categories. For example, the first 500 floors will be for food, medical, shops, while the rest of the floor is for the businesses only. I have separated the color of the elevator, the first 500 floor will be color silver while the 501th floor to 1000th is teal. I’ve also separated the going up and going down elevator, where it will lessen the people’s waiting time on the elevator.

In this way, the line for the users of elevator will lessen. Also, I would require 15,000 pounds for the capacity of the elevator.

Inside the elevator, there will be two screens and two touch screens on both sides of the elevator’s door. The screen was located on top both sides, it will appear what floor you’re into, the capacity of it. While the touch screen contains the number 0-9, the backspace and the letter B and P. B stands for basement while P stands for Penthouse. It will also appear the inputter number and what number will you input.

The travel time from 1st to 500th floor will be 7minutes without interruptions. I’ve went to Strata 2000 building which has a 28istorey, and out of curiosity experience the 17th floor and the travel time is only 21 seconds without interruptions.
